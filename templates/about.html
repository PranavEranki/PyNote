Keep a note. Keep ideas. Keep your dreams.

PyNote is a simple note taking Django project. It was developed to act as a hands-on project to increase my MTV framework knowledge.



    Features of PyNote - keep a Note

  • Customize notes using the markdown editor. You dont need to know markdown syntax, as the editor will help you.
  • Save, Update, delete notes, and assign tags.
  • Support for code blocks and highlighting.
  • Fully responsive. Usable on any site, once published, and view automatically adjusts.
